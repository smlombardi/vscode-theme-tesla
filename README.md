
theme-tesla
==========

Origially created for Atom, this is a relaxing but colorful dark syntax color for Microsoft VSCode.

Optimized for front-end development, SCSS, CSS, HTML, PHP,  JS, and JSON

Update 5.0:  Added Typescript support
